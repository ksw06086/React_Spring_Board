react 폴더 기능
- apis : backend와 연결할 AXIOS(http 통신) 함수들
- assets : 이미지, 비디오, 폰트
- components : 컴포넌트 저장(최소 단위<ex 버튼, 카드, 인풋>)
- constants : 상수들
- hooks : hook 함수들
- layouts : 헤더나 풋 같은 것들
- mooks : 쓸 일이 없을 것 같은데 넣어둠
- stores - 전역 상태 관리
- types - 각종 Type 저장
- utils - 각종 메서드들
- views - 컴포넌트 저장(큰 페이지 혹은 레이아웃 단위<ex 로그인, 메인, 게시물리스트, 마이페이지, top3 뷰, 최신게시물 뷰, 인기검색어>)

html은 한 페이지 단위로 랜더링됨
- 문제점 : 동영상 시청 중 좋아요 버튼 클릭 시 숫자 상태 증가 시 랜더링되어 동영상도 재랜더링이 되니 계속 볼 수 없고 처음부터 봐야함
- 해결 : 
1) 좋아요 버튼과 숫자 컴포넌트, 동영상 시청 컴포넌트로 나누어서 관리
2) 좋아요 버튼 클릭시 관련 컴포넌트만 재랜더링 시켜줌
* 재랜더링 기준이 state(해당 state를 사용하는 컴포넌트만 재랜더링이 됨)
